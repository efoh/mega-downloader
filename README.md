# Mega.nz Downloader

A simple Python script to download and decrypt files from Mega.nz links.  

## Features

- Download files from Mega.nz
- Decrypt encrypted files
- Automatically extract ZIP, RAR, and GZIP archives
- Saves files to `downloads` folder

## Requirements

- Python 3.8+
- Modules:
  - `requests`
  - `pycryptodome`

Install modules:

```bash
pip install requests pycryptodome
